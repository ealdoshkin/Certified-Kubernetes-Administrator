# Tools Documentation

This directory contains documentation for useful tools used in Kubernetes administration and CKA exam preparation.

## Available Tools

- `bash.md` - Bash scripting tips and tricks
- `jq.md` - jq JSON processor documentation
- `yq.md` - yq YAML processor documentation
- `vim.md` - Vim editor tips and shortcuts
- `tmux.md` - tmux terminal multiplexer guide

## Why These Tools?

These tools are essential for:
- **bash**: Shell scripting and automation
- **jq**: Processing JSON output from kubectl commands
- **yq**: Processing and editing YAML files
- **vim**: Quick file editing in exam environment
- **tmux**: Managing multiple terminal sessions

Mastering these tools will significantly improve your efficiency during the CKA exam.
