# Examples

This directory contains example Kubernetes manifests and troubleshooting scenarios.

## Structure

- `troubleshooting/` - Common error scenarios and their solutions
  - `crashloopbackoff.yml` - CrashLoopBackOff error example
  - `imagepullerror.yml` - ImagePullError example
  - `resourcepod.yml` - Resource-related pod issues

## Usage

These examples help you understand common Kubernetes issues and how to troubleshoot them, which is crucial for the CKA exam (30% of the exam focuses on troubleshooting).
