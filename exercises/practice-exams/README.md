## Format

Each practice exam contains:
- Multiple choice questions
- Hands-on scenarios
- Answer explanations
- kubectl command examples

## How to Use

1. **Simulate exam conditions**:
   - Set a 2-hour timer
   - Use only allowed resources (kubectl, documentation)
   - Complete questions in any order

2. **Review answers**:
   - Check explanations for each question
   - Understand why answers are correct
   - Practice the commands shown

3. **Track progress**:
   - Mark questions as "Mastered" or "Not Mastered"
   - Focus on weak areas
   - Retake exams to measure improvement
