# Kubernetes Resource Templates

This directory contains base YAML templates for common Kubernetes resources.

## Available Templates

- `pod.yml` - Pod template
- `deploy.yml` - Deployment template
- `rs.yml` - ReplicaSet template
- `ds.yml` - DaemonSet template
- `job.yml` - Job template
- `cj.yml` - CronJob template
- `svc.yml` - Service template
- `sfs.yml` - StatefulSet template
- `ns.yml` - Namespace template
- `configmap.yml` - ConfigMap template
- `secret.yml` - Secret template
- `crd.yml` - CustomResourceDefinition template
- `cr.yml` - CustomResource template

## Usage

These templates can be used as starting points for creating Kubernetes resources. Customize them according to your needs.
